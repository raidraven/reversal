# REVERSAL 収益計画書(月1万円達成 90日ロードマップ)

作成日: 2026-07-15
前提: 作業時間 1日1〜2時間(週10時間)。開発系タスクはClaude Codeに依頼する前提の見積もり。

## 収益モデルの見立て

90日で検索流入だけで月1万は困難。現実的な構成:

> **月1万円 = 高単価アフィリ発生2件(6,000円)+ 有料note 4部(3,000円)+ AdSense(1,000円)**

逆算: アフィリ発生2件 ← クリック60回 ← 収益記事PV 2,000 ← サイト全体PV 6,000 ← **記事20本 × Xフォロワー500人**

※セルフバックは初日ブースト用(一回性)。「事業の初期資金調達」と位置づける。

## 診断サマリー

### 強み
- 世界観の独自性(洋館×仮面舞踏会のブランディング)
- 技術実装の完成度(ゲーミフィケーション・AI執事・モデレーション実装済み)
- クロエ=ユーザー状況を踏まえて応答する競合にない機能
- 低ランニングコスト・運営者の開発力

### 弱み(深刻度順)
- ★★★ コンテンツ資産ゼロ(SEOの土俵に立っていない)→ 書庫機能で対応済み(2026-07-15)
- ★★★ 収益導線ゼロ(アフィリ0・広告0・決済なし。premiumUnlocked未使用)
- ★★★ 独自ドメインなし(vercel.appサブドメイン)
- ★★ 実績・活気の不在(登録1名)/ 閉じた設計 / 運営者情報なし

### 戦略の核
- 検索で競合と正面勝負せず、まず**X→サイト**の流入を作る
- 記事は「継続」軸のロングテールKW(「AI副業 続かない」「副業 三日坊主 対策」等)
- 差別化軸は**誠実さ(収益0円からの実況型)**。煽り系競合との対比
- E-E-A-Tは実況型で「経験」を積み上げて担保する
